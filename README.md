# hubot-doyoubi

A hubot script for Doyoubi!!

See [`src/doyoubi.coffee`](src/doyoubi.coffee) for full documentation.

## Installation

In hubot project repo, run:

`npm install hubot-doyoubi --save`

Then add **hubot-doyoubi** to your `external-scripts.json`:

```json
[
  "hubot-doyoubi"
]
```

## Sample Interaction

```
user1>> hubot hello
hubot>> hello!
```
